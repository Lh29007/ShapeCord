## ⚡ What is ShapeCord?

ShapeCord is a **lightweight, optimized and clean Discord client wrapper** built with Electron.  
It loads the official Discord web app, but with:

- 🚀 Faster startup  
- 🧠 Lower RAM usage  
- 🎯 No useless animations  
- 🛡️ No mods, no injections, 100% safe  
- 🖥️ Native Windows `.exe` installer  

---

## ✨ Features

- ⚡ **Fast launch** thanks to optimized Electron flags  
- 🧼 **Cleaner UI** (reduced animations, smoother experience)  
- 🔒 **No modifications to Discord** (safe for your account)  
- 🧊 **Low resource usage**  
- 🪟 **Windows installer included**  
- 🎨 **Custom icon & branding**

## 🛠️ Installation

1. Download `ShapeCord Setup.exe`  
2. Run the installer  
3. Launch ShapeCord from your desktop  
4. Log in to Discord  
5. Enjoy 🎉  

---

## 🧩 How it works

ShapeCord uses:

- **Electron** to wrap the Discord web app  
- A custom preload script to reduce animations  
- Optimized flags to reduce RAM usage  
- A clean UI with no devtools, no menus, no distractions
